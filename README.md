# Simplilearn-Capstone-Project

## Fashionista

Fashionista contains the angular project.  
Run: 
1. npm install
2. ng serve -o  
To launch.

## Fashionista-Server

Fashionista-Server contains the backend needed for the Fashionista angular project to work.  
Run:
1. npm install
2. nodemon server.js (or node server.js)
